# TeamTask — Fullstack Task Management App

TeamTask adalah aplikasi task management fullstack untuk membantu tim kecil mengatur project, task, assignment, deadline, dan status pekerjaan.

Project ini dibuat sebagai portfolio fullstack untuk menunjukkan kemampuan membangun aplikasi web end-to-end mulai dari frontend, backend, database, authentication, API, deployment, hingga integrasi Telegram notification.

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod
- Axios / TanStack Query

### Backend
- Laravel
- PostgreSQL
- Laravel Sanctum

### Deployment
- Frontend: Vercel
- Backend: Railway / Render
- Database: Supabase PostgreSQL / Railway PostgreSQL

### Planned Integration
- Telegram Bot Notification

## Main Features

- Authentication
- Role Admin & Member
- Project Management
- Task Management
- Task Assignment
- Status Tracking
- Dashboard Statistics
- Search & Filter
- Task Comments
- Activity Log
- Telegram Notification
