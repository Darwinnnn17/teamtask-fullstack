# TeamTask — Fullstack Task Management App

TeamTask adalah aplikasi task management fullstack untuk membantu tim kecil mengatur project, task, assignment, deadline, dan status pekerjaan.

Project ini dibuat sebagai portfolio fullstack untuk menunjukkan kemampuan membangun aplikasi web end-to-end mulai dari frontend, backend, database, authentication, API, deployment, hingga integrasi Telegram notification.

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- PostgreSQL
- Prisma ORM

### Authentication
- JWT Authentication
- bcrypt password hashing

### Validation
- Zod

## Main Features

- Authentication
- Role Admin & Member
- Project Management
- Task Management
- Task Assignment
- Status Tracking
- Dashboard Statistics
- Search & Filter
- Task Comments
- Activity Log
- Telegram Notification
